## Reddit — r/ClaudeAI

**Title:** Built with Claude Code: an open-source framework for Google Stitch MCP

**Body:**

Sharing a project I built entirely with Claude Code — **Stitch Forge**, a CLI framework for automating web design with Google Stitch.

It ships with 6 slash commands:
- `/forge-design` — Generate a design system
- `/forge-generate` — Create screens with AI
- `/forge-build` — Export to Static HTML, Astro, or Next.js
- `/forge-preview` — Preview inline via MCP
- `/forge-research` — Check for Stitch updates
- `/forge-sync` — Pull from Stitch projects

The whole repo (38 tests, TUI, adapters, research module) was built in one session using Claude Code with worktree agents for parallel feature development.

**Proof of concept**: https://freptar0.github.io/stitch-forge/ — this landing page was generated by the tool itself.

**Repo**: https://github.com/FReptar0/stitch-forge
