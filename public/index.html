How It Works

From idea to production in three steps.

1

Describe your brand

$ forge design
"Acme Corp, SaaS, startups, modern minimal"
→ DESIGN.md created (8 sections)

A single command generates your full design system — colors, fonts, spacing, and component patterns.

2

AI generates screens

$ forge generate
"Landing page with hero, features, CTA"
→ screens/homepage.html saved
→ Preview opened in browser

Stitch MCP builds pixel-perfect HTML from your prompt, guided by your DESIGN.md rules.

3

Export & deploy

$ forge build --framework nextjs
→ out/app/page.tsx
→ out/app/about/page.tsx
Site built successfully!

Export to Static HTML, Astro, or Next.js. Deploy anywhere — Vercel, Netlify, GitHub Pages.
